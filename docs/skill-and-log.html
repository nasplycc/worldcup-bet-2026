# WorldCup Bet 2026 赛事分析技能 Python 只负责整理赛程、赔率和候选方向； 最终 AI 判断由大模型自身完成。 ## 读取数据 - home_team / away_team - kickoff_time - recommendations.candidate_bets - recommendations.team_profiles - recommendations.search_queries - recommendations.odds_movement ## 分析任务 结合联网搜索和自身模型分析： - 球队实力、排名、近期状态 - 伤病、停赛、阵容、主教练战术 - 小组形势、赛程密度、轮换动机 - 比赛地点、气候、旅行距离 - 市场数据回报、让球盘、冷门概率 ## 动态候选池 候选池不是静态结论。若触发刷新条件， 必须重新排序或剔除候选： - 竞彩赔率或让球数变化 - 赛前首发阵容确认 - 关键球员伤停、停赛 - 小组积分和出线形势变化 - 距离开赛 24h / 6h / 1h 例行刷新 ## 目标风格 高风险高回报，优先找赔率价值 和可搏冷门，不做保守稳胆。 ## 输出格式 - 最佳下注：玩法 / 号码 / 赔率 - 下注话术 - 为什么值得搏 / 主要风险 - 放弃条件 - 建议注码：小注 / 搏冷小注 / 放弃 - 高赔备选（2-3个）
